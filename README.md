# Wellspring

Wellspring is a multi-tenant content management platform for wellness creators.

Creators can manage branded programs, organize audio/video sessions, upload media, import session data from CSV, and review administrative audit history.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Node.js
- Express
- PostgreSQL hosted on Supabase
- AWS S3 pre-signed uploads

## Repository Structure

apps/web        Next.js admin panel
apps/api        Express API
packages/shared Shared schemas and types
docs            Project documentation
ai-history      AI development history
